## 简介

JsonWeaver 是一个基于 [Jolt](https://github.com/bazaarvoice/jolt) 引擎构建的可视化 JSON 结构转换工具，通过直观的界面将一种 JSON 格式转换为另一种 JSON 格式。支持实时预览、主题切换和示例演示，适用于 API 数据适配、配置转换等场景。

## 核心功能

✨ **JSON 格式双向转换**  
✨ **可视化配置编辑器**  
✨ **实时转换结果预览**  
✨ **深色/浅色主题切换**  
✨ **内置示例演示系统**  
✨ **支持复杂嵌套结构处理**  
✨ **基于 Jolt 引擎的强大转换能力**
